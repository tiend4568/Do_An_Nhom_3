# Du An Nhom - Bai Thuc Hanh So 3

## Mo ta
He thong Quan ly Sinh vien don gian viet bang Java.  
Du an duoc thuc hien trong khuon kho Bai thuc hanh so 3 mon **Software Configuration Management** tai Dai hoc Duy Tan.

## Cau truc du an
```
du-an-nhom-btth3/
├── src/main/
│   ├── Main.java              ← Chuong trinh chinh
│   ├── SinhVien.java          ← Lop doi tuong SinhVien
│   └── QuanLySinhVien.java    ← Lop quan ly danh sach
├── docs/
│   └── tailieu.docx           ← Tai lieu mo ta du an
└── README.md
```

## Chay chuong trinh
```bash
javac src/main/*.java
java -cp src/main Main
```
